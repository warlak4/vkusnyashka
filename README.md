# Кондитерская «Вкусняшка»

Онлайн-магазин кондитерских изделий с возможностью:
- Просмотра каталога товаров
- Создания собственного торта
- Оформления заказа с доставкой
- Отслеживания статуса заказа
- Оставления отзывов

## Технологии
- HTML5
- CSS3
- JavaScript
- Яндекс Карты API
